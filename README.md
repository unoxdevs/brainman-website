# 🖥️ Brainman Website
A nice and minimalist website for Brainman.

## 🧰 Installation
- Download the project files to your own computer.
- Open the terminal and type “npm install” then “npm run dev” to open the site.
**Note:** There is an .env file, enter the url address of your Pocketbase hosting here.
