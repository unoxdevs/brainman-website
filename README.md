# Brainman Website

This project is a simple AI website built with **SvelteKit** and **TailwindCSS**.

## Features

- **Interactive Chat Interface**: A user-friendly chat interface where users can send messages and receive responses.
- **Code Block Formatting**: The chatbot can format code blocks and inline code to display them in a readable manner.
- **Predefined Actions**: Buttons for predefined actions like generating Python code, explaining concepts, and summarizing text.
- **Responsive Design**: The chat interface is fully responsive, optimized for desktop, tablet, and mobile views.
- **Thinking Indicator**: The chatbot shows a "Thinking..." message while processing the user's input.
- **Message Scrolling**: Automatically scrolls to the latest message in the conversation.

## Technologies Used

- **SvelteKit**: A framework for building modern web applications with a focus on performance and simplicity.
- **TailwindCSS**: A utility-first CSS framework for designing responsive and modern user interfaces.
- **Axios**: A promise-based HTTP client for making API requests.

## Project Setup

### Prerequisites

Before you begin, ensure you have the following software installed:

- **Node.js** (version 16 or later)
- **npm** (or **yarn** as an alternative)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/unoxdevs/brainman-website.git
   cd brainman-website
   ```

2. Install the dependencies:

   ```bash
   npm install
   ```

3. Run the project:

   ```bash
   npm run dev
   ```
